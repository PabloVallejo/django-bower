Bower In Your Django Projects.

Bower is an easy to use Front-End Package manager, even though its popularity it's not very used in Django projects.

This page describes how to use Bower in Django using Django Bower create by Vladimir Iakovlev

View Working Project on GitHub

Adding Dependencies

In your settings.py file describe the dependencies you want to use.

# Project dependencies
BOWER_INSTALLED_APPS = (
    'jquery#1.9',
    'bootstrap',
)

Installing Dependencies

In order to install the dependencies you we have in settings.py we need to run bower install using manage.py.

./manage.py bower install

Including dependencies using Pipeline

In settings.py file describe the dependencies you want to use and have compressed.

PIPELINE_CSS = {
    
    # Libraries
    'libraries': {
        'source_filenames': (   
            'bower_components/bootstrap/dist/css/bootstrap.css',
        ),
        'output_filename': 'css/libs.min.css',
    }
}

Template Inclusion

Include dependencies in your template using pipeline compressed.

{% load compressed %}

...

<!-- Include pipeline dependencies -->
{% compressed_css 'libraries' %}

...

Jenkins Deploy

In your jenkins.sh build file, make sure that bower components get fetched before statics are collected.

#...

# Install bower 
./manage.py bower install

# Collect static files
./manage.py collectstatic --noinput --ignore=*.sass

#...

.gitignore configuration

As bower components get fetch on build, we don't need to have them in our repository.

# Ignore bower components
/app/static/bower_components/
